# Bank Foundry

Bank Foundry is a modular platform comprising a robust FastAPI backend for merchant and bank operations along with a responsive Vite + React frontend.

## Prerequisites
- **Python 3.10+**
- **Node.js 18+**
- **PostgreSQL**

## Project Structure
- `app/`: Python backend source code, built as a FastAPI application (`app.api.server`).
- `config.py`: Configuration and environment settings for the backend.
- `frontend/`: The Vite + React single-page application.
- `tests/`: Automated test suite for validating core backend functionality.

## Setup Instructions

### 1. Database Setup
Ensure PostgreSQL is installed and running. By default, the application connects to a local database with credentials defined in `config.py`.
If you are running a different database setup, you can add a `.env` file in the root of the project with your database overrides:
```env
DATABASE_URL=postgresql://user:password@localhost:5432/your_database
```

### 2. Backend Setup
The backend is a traditional Python project focusing on FastAPI.
1. Configure a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Set up environment variables. You will need external API keys, especially for the LLM integration. Create a `.env` in the root:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
3. Install standard ASGI dependencies alongside any specific packages required by the app (FastAPI, uvicorn, SQLAlchemy, Langchain, etc.).
4. Run the backend server using Uvicorn:
   ```bash
   uvicorn app.api.server:app --host 0.0.0.0 --port 8000 --reload
   ```

### 3. Frontend Setup
The frontend is a Vite + React application.
1. Navigate to the `frontend/` directory:
   ```bash
   cd frontend
   ```
2. Install the necessary Node dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Local Development
With both servers running, the FastAPI backend will be serving endpoints at `http://localhost:8000`. You can also visit the autogenerated Swagger API docs at `http://localhost:8000/docs`. The Vite frontend will typically be accessible at `http://localhost:5173`. Make sure the frontend `.env` points correctly to the FastAPI backend API path if defaults need overriding.
